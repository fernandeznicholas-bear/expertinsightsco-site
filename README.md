# expertinsightsco.com

The Expert Insights Co. company website. Hand-built static site — no framework, no platform fees.

- **Hosting:** GitHub Pages (free), custom domain `expertinsightsco.com`, TLS automatic.
- **Deploys:** push to `main` → live in ~1 minute.
- **Brand:** Navy `#0B1D3A` / Orange `#F47C20`, Montserrat — per the brand kit in the Expert Insights operating system (`Expert Insights Claude Coach/Storefront Coach Code/BRAND.md`).
- **Content source:** `EXPERT_INSIGHTS_SITE.md` in the operating-system folder (6 pages of approved copy).
- **Plan & phases:** see `EXPERT_INSIGHTS_SITE.md` header + `TASKS.md` in the operating-system folder.

This repo is deliberately OUTSIDE the Google Drive folder — git repos inside Drive sync risk corruption (same rule as `melania-theme`).
